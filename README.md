## Alex Benny — AI Systems Portfolio

This is a minimalist, production-ready single-page portfolio for **Alex Benny**, focused on deterministic AI systems, agentic architectures, and execution layers.

### Stack

- **Build tool**: Vite (React)
- **UI**: React 18 + Tailwind CSS
- **Animation**: Framer Motion

### Development

- `npm install`
- `npm run dev`
- Visit `http://localhost:5173`

### Production

- `npm run build`
- `npm run preview` to verify the static build locally.
